# m3u-radio-stations-playlist-for-vlc
# Lista radio stanica koja se lako može otvoriti VLC-om ili AIMP-om na mobilnom i onda lako birati stanice. Uglavnom rok i pop stanice, većina su čiste Internet stanice i nema reklama.

List of some radio stations streams as .M3U file that can be opened with VLC or any other player (I use _AIMP_) on the phone and easily switch between the stations

The list contains mainly rock stations from Serbia, Srpska, Austria, Germany, France and Norway, but also some Pop, Jazz and Classical.

Most of the stations are pure Internet stations and are Ad-free, like Naxi, Big Rock.
